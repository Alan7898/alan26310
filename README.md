# 智能中式海报生成器 - 移动端版本

一个基于AI的智能海报生成器，专为移动端优化设计。

## 功能特点

- 🤖 **AI智能分析**：自动分析输入内容，生成结构化的海报信息
- 🎨 **中式风格设计**：传统中国风UI设计，典雅大气
- 📱 **移动端优化**：完美适配手机浏览器
- 🖼️ **图片上传**：支持上传自定义图片到海报
- 💾 **一键下载**：生成JPG格式海报并下载到本地
- 🌐 **跨浏览器兼容**：支持iOS Safari、Android Chrome、微信浏览器等

## 文件说明

- `poster-generator-mobile.html` - 标准移动端版本（使用阿里云OSS图片）
- `poster-generator-mobile-embedded.html` - 自包含版本（CSS备用方案）
- `poster-generator-wechat.html` - 微信专用优化版本

## 使用方法

1. 在移动端浏览器中打开HTML文件
2. 输入任意内容
3. 点击"智能分析并生成海报"按钮
4. 可选择上传图片
5. 点击"下载海报"保存到相册

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript (ES6+)
- html2canvas (海报生成)
- 智谱AI API (内容分析)

## 部署说明

直接部署到任意静态文件服务器即可，无需后端支持。

## 注意事项

- 需要网络连接以调用AI API
- 图片上传功能需要浏览器支持File API
- 下载功能在部分浏览器可能需要手动授权
