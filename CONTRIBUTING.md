# 贡献指南

感谢您对智能中式海报生成器项目的关注！

## 如何贡献

1. **Fork** 本项目
2. **创建分支** (`git checkout -b feature/AmazingFeature`)
3. **提交更改** (`git commit -m 'Add some AmazingFeature'`)
4. **推送分支** (`git push origin feature/AmazingFeature`)
5. **创建 Pull Request**

## 报告问题

如果您发现了bug或有功能建议，请通过 GitHub Issues 提交。

## 代码规范

- 保持代码简洁清晰
- 添加必要的注释
- 确保移动端兼容性
- 测试所有主要浏览器

## 联系方式

如有问题，欢迎通过 GitHub Issues 联系。
